## ğŸ“ Support & Maintenance

### Common Issues

**Forms not sending emails?**
- You'll need a backend service to handle form submissions
- Use services like Formspree, Netlify Forms, or Firebase

**Logo not showing?**
- Ensure logo.png is in the assets folder
- Check file path in HTML

**Responsive design not working?**
- Clear browser cache
- Check viewport meta tag is present
- Test in incognito/private window
