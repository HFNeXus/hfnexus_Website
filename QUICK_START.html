üìû Setting Up Contact Forms

Important!

Static GitHub Pages don't handle form submissions directly. You need a backend service:

  1. Formspree (Recommended)
    • Sign up: https://formspree.io
    • Update form action in contact.html
    • Free tier available
  2. Netlify Forms
    • Host on Netlify instead of GitHub Pages
    • Automatic form handling
  3. Basin
    • Simple form backend: https://www.basinforms.com
